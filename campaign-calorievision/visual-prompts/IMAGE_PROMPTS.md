## PRODUCTION CHECKLIST

Before using these prompts:
- [ ] Open Google Gemini / ImageFX
- [ ] Paste the full prompt for the desired size
- [ ] Generate 3–4 variations and select the best
- [ ] Download as high-resolution JPG or PNG
- [ ] Rename: `awareness.jpg` / `split-a.jpg` / `split-b.jpg`
- [ ] Place in: `campaign-calorievision/images/`
- [ ] Add text overlays in Meta Ads Manager or Canva
- [ ] Check that no phone UI or text is visible in the base image
- [ ] Confirm image is at minimum 1080px wide

**If the generated image shows phone UI or text on screen:**
→ Add to your prompt: "The phone screen is completely black or off — only the physical phone body is visible, no screen content"

**If the person's face looks unnatural:**
→ Add: "natural, candid facial expression, genuine emotion, not posed"

**If the background is too busy:**
→ Add: "very clean minimal background, shallow depth of field, blurred background"
